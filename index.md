# Dokumenation der Monitor API

Die Kommunikation via **HTTPS** erfolgt mit den Bibliotheken **[JQuery](https://jquery.com/){:target="_blank"}** und **[Axios](https://github.com/axios/axios){:target="_blank"}**.
